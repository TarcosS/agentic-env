# No AI contribution signatures

Never add AI attribution to anything:
- No `Co-Authored-By: Claude` (or any AI) trailer.
- No `🤖 Generated with Claude Code` footer.
- No "generated by AI" / "created with AI" notices anywhere.

Applies to commit messages, PR titles and bodies, and any other output.

Backstop: `.git/hooks/commit-msg` strips these from commit messages automatically (local, not shared — non-AI contributors don't need it).
